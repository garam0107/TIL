### 스택의 push 알고리즘
```python
def push(item, size):
    global top
    top += 1
    if top == size:
        print('overflow!')
    else:
        stack[top] = item

size = 10
stack = [0] * size
top = -1

push(10,size)
top += 1   #push 20
stack[top] = 20
```
