### HTML form
- action
  - 입력 데이터가 전송될 URL을 지정(목적지)
  - 만약 이 속성을 지정하지 않으면 데이터는 현재 form이 있는 페이지의 URL로 보내짐
- method
  - 데이터를 어떤 방식으로 보낼 것인지 정의

- name 
  - input의 핵심 속성
  - 사용자가 입력한 데이터에 붙이는 이름(key)
- Query String Parameters
